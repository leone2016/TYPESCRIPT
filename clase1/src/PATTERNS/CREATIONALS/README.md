# PATTERNS CREATIONALS

- create object without specifying the exact class of object that will be created

## What do they solve ? 
- How to create objects efficiently
- Hide the logic of object creation from the client code
- Reduce coupling between objects (Classes)
- Improve flexibility and scalability of the application
- Promote single responsibility principle

## Examples
- Factory Method - Factory Function
- Abstract Factory
- Builder
- Prototype
- Singleton
- Inmutability with copy
